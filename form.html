HTML Form Basics

HTML forms are used to collect user input. Forms are an essential part of web development, allowing users to interact with your website and provide information.

Basic Form Elements

  • <form> - Defines the form
  • <input> - Input field for user data
  • <label> - Label for form elements
  • <select> - Dropdown list
  • <textarea> - Multi-line text input

Here is an example of a simple form structure:

<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <br><br>
  
  <label for="email">Email:</label>
  <input type="email" id="email" name="email">
  <br><br>
  
  <input type="submit" value="Submit">
</form>

HTML Form with Validation

HTML5 provides form validation with various input types like email and password, ensuring that the user provides valid data.
