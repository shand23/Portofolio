Tutorial Deploy Github

Step 1: Create a GitHub Repository

1. Open GitHub and log into your account.

2. Click the New button to create a new repository.

3. Name the repository (e.g., my-tutorial-website).

4. Set the repository to Public and click Create repository.

Step 2: Upload Your Website Code to the Repository

There are two ways to upload files:

  • Direct upload on GitHub:

    • 1. In the new repository, click Add file > Upload files.

    2. Drag and drop your HTML, CSS, JavaScript, and image files into the repository.

    3. When done, click Commit changes to upload the files.

  • Upload using Git in the terminal:

    • 1. Open the terminal and navigate to your project directory.

    2. Run the following commands to initialize Git, add files, and push them to GitHub:

    git init
    git add .
    git commit -m "Initial commit"
    git branch -M main
    git remote add origin https://github.com/username/my-tutorial-website.git
    git push -u origin main

Step 3: Enable GitHub Pages

1. Open the Settings tab in your GitHub repository.

2. Scroll down to the GitHub Pages section.

3. In the Source option, select main and the folder / (root).

4. Click Save.

Your website URL will appear in the GitHub Pages section. The URL typically looks like https://username.github.io/my-tutorial-website.

Step 4: Check Your Website

Open the generated URL to view your live website!
