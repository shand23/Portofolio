HTML Table Basics

HTML tables allow web developers to arrange data into rows and columns.

Simple HTML Table

A basic HTML table consists of the following elements:

  • <table> - Defines the table
  • <tr> - Defines a table row
  • <th> - Defines a header cell
  • <td> - Defines a table data (cell)

Here is an example of a simple HTML table:

<table>
        <tr>
            <th>Name</th>
            <th>Age</th>
        </tr>
        <tr>
            <td>John</td>
            <td>25</td>
        </tr>
        <tr>
            <td>Doe</td>
            <td>22</td>
        </tr>
    </table>

HTML Table with Borders

You can add borders to your table using the border attribute:

<table border="1">
        <tr>
            <th>Name</th>
            <th>Age</th>
        </tr>
        <tr>
            <td>John</td>
            <td>25</td>
        </tr>
        <tr>
            <td>Doe</td>
            <td>22</td>
        </tr>
    </table>

HTML Table with Styling

You can style your HTML tables using CSS. Here is an example:

table {
        border-collapse: collapse;
        width: 100%;
    }
    th, td {
        border: 1px solid black;
        padding: 8px;
        text-align: left;
    }
